# 苔藓名称导航系统 v1.0.0

**首次正式发布** | 2026-07-01

---

## 📖 项目简介

苔藓名称导航系统是一个基于 Python + Flask + SQLite 的本地检索工具，用于按苔藓名称快速定位多本书籍中的页码，并集成苔藓名称数据库（来源：sp2000.org.cn）。

**核心用途**：帮助苔藓研究者、植物学爱好者和学生快速查找苔藓名称在各类专业书籍中的位置。

---

## ✨ 核心功能

### 1. 智能检索
- **模糊查询**：输入苔藓名称的一个字或一个词即可检索
- **支持全层级**：纲/目/科/属/种等层级名称均可搜索
- **多书籍聚合**：同名苔藓在不同书籍的页码统一展示
- **书籍过滤**：可按所选书籍过滤显示结果

### 2. 浏览功能
- **按书籍浏览**：选择某一本书，查看其下所有苔藓名称及页码
- **苔藓名称数据库**：浏览门→纲→目→科→属→物种级联结构
- **苔藓名称脑图**：可视化展示苔藓名称的分类层级关系

### 3. 数据管理
- **数据导入**：支持上传 xlsx 文件批量导入书籍页码数据
- **重复导入**：可选择导入前清空旧数据
- **灵活配置**：支持多书籍管理和PDF页码换算

## 🗂️ 数据库结构

系统使用 SQLite 数据库，包含以下表：

### books 表（书籍信息）
- `id` - 书籍ID
- `book_name` - 书籍全称
- `short_name` - 书籍简称
- `offset` - PDF页码偏移量
- `has_pdf` - 是否有PDF（1有 0无）
- `region` - 地区
- `publisher` - 出版社
- `publish_date` - 出版日期
- `price` - 价格
- `page` - 页数
- `moss_count` - 苔藓数量
- `author` - 作者
- `cover_image` - 封面图片
- `notes` - 备注
- `has_txt` - 文字描述（1有 0无）
- `has_line` - 线条图（1有 0无 2部分有）
- `has_spec` - 标本图（1有 0无 2部分有）
- `has_env` - 生境图（1有 0无 2部分有）
- `has_micro` - 显微图（1有 0无 2部分有）
- `has_micro_section` - 显微剖面（1有 0无 2部分有）
- `has_elec` - 电镜图（1有 0无 2部分有）

### mosses 表（苔藓名称）
- `id` - 苔藓ID
- `moss_name` - 苔藓名称

### moss_pages 表（页码数据）
- `id` - 记录ID
- `book_id` - 书籍ID（外键）
- `moss_id` - 苔藓ID（外键）
- `page_id` - 页码

### moss_data 表（苔藓名称数据库）
- 包含门、纲、目、科、属、物种的拉丁名和中文名

**数据库位置**：
- 开发版：`项目根目录/booknavi.db`
- 桌面版：`exe同目录/booknavi.db`

可使用 Navicat 等数据库工具查看和管理数据库，账号密码为空。

## 📥 数据导入格式

### 书籍页码数据（xlsx）

**文件结构**：
- 每个 sheet 代表一本书
- 第1列：苔藓名称
- 第2列：页码

**导入步骤**：
1. 访问"查询"页右上角按钮"导入书籍页码表格"（或访问 `/importpages`）
2. 上传 xlsx 文件
3. 选择是否"导入前清空旧数据"
4. 点击导入

**注意事项**：
- 导入时自动去除左右空格
- 空行自动忽略
- 支持重复导入（可选择清空旧数据）

### 书籍目录数据（xlsx）

**用途**：导入书籍元数据（书名、作者、出版社等）

**导入步骤**：
1. 访问"书籍目录"页（或访问 `/importbooks`）
2. 上传 xlsx 文件
3. 选择是否"导入前清空旧数据"
4. 点击导入

**xlsx 表头**：需与 `books` 表列名一致（参见数据库结构）

## 🔧 技术栈

- **后端**：Python 3.12 + Flask
- **数据库**：SQLite 3
- **前端**：HTML + CSS + JavaScript
- **桌面版 GUI**：pywebview 5.4（内嵌 WebView2 引擎）
- **WebView2 Runtime**：Windows 10 1803+ 自带 Chromium 引擎
- **.NET**：pythonnet + clr-loader 调用 WebView2 COM 接口
- **打包工具**：PyInstaller 6.x
- **依赖库**：
  - Flask（Web框架）
  - openpyxl（Excel文件处理）
  - pypinyin（拼音排序）
  - pywebview（桌面窗体）
  - pythonnet（.NET调用）

## 📝 使用说明

### 查询页（/）
- 输入关键词检索苔藓名称
- 勾选要显示的书籍
- 结果按书籍分组展示
- 显示换算后的 PDF 页码（如已配置偏移量）

### 按书籍浏览（/browse）
- 选择一本书
- 浏览其下所有苔藓名称及页码

### 苔藓名称数据库（/taxonomy）
- 浏览门→纲→目→科→属→物种级联结构
- 支持模糊匹配和分页

### 苔藓名称脑图（/mindmap）
- 可视化展示苔藓名称的分类层级关系

### 书籍目录（/catalog）
- 查看所有书籍信息
- 支持按地区、特色内容等条件过滤
- 支持按出版日期排序

## ⚠️ 已知问题

1. **桌面版首次启动较慢**：需要复制数据库到 exe 同目录，请耐心等待
2. **WebView2 依赖**：部分精简版 Windows 可能缺少 WebView2 Runtime，需手动安装
3. **数据导入失败**：确保 xlsx 文件格式正确，表头与数据库列名一致
